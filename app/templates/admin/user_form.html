How Authentication Works
  1. Admin creates the user here with their AD username and assigns a role.
  2. User goes to the login page and enters their AD username and password.
  3. KeyVault authenticates against Active Directory via LDAP.
  4. On successful login, user details are synced from AD and the assigned role determines their permissions.
Roles:
  • admin Full access + user/group management
  • user Create, edit, delete, share own secrets
  • readonly View shared secrets only
